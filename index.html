Recursos Humanos

Um sistema de gestão de recursos humanos (SGRH) é um software que automatiza as tarefas e processos relacionados à gestão de pessoas em uma organização. Para o desenvolvimento deste, foi usado o editor do codigo Visual Studio Code, PHP, CSS e MysQL

Impostos Chongoene

Sistema de controlo de pagamento de impostos, este sistema permite que as autoridades fiscais controlem de forma eficaz os pagamentos de impostos de todos os indivíduos que vivem no distrito. Para este usou-se JavaScript, CSS, MySQL, VS Code, PHP e HTML
